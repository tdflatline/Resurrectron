Resurrectron

May all the twitternets be drowned by the din of inhuman laughter.

Resurrectron is a project to create a twitter+foursquare AI based on a single person's tweets, blog posts, mailinglist posts, irc logs, foursquare data, and other written material.

The AI uses an HMM trained on part of speech Ngrams as a grammar model to generate novel text based on the input corpus, and uses an augmented TF-IDF search to respond to messages and interesting tweets by people that the AI follows.

It can also operate in a pure quotation engine mode without the HMM, and will use TF-IDF to respond to messages with direct quotes from a quote pool in this mode.

It has a number of additional machine learning techniques to normalize and de-normalize highly colloquial twitter text, to cluster documents/tweets by topic for better HMM training, to correlate concepts of tweets and messages using Wordnet's hyponyms, hypernyms, and antomyms, and to remember the conversation context of each user that messages it.

For more information on its current capabilites see the current settings file.

Eventually it will also check in to foursquare and tweet topically about its current location. For more future plans, see the TODO file.

Dependencies

  1. nltk 2.08b
  2. Nodebox linguistics 1.9.4.2
  3. Optional, but recommended: AGFL 2.8 and CDL3 1.2.7
